# Aditya Solanki — Developer Portfolio

Live at → **https://aditya018s.github.io**

## Stack
Pure HTML · CSS · Vanilla JS — no build step, no npm, opens directly in browser.

## Structure
```
Portfolio/
├── index.html        ← All sections
├── styles.css        ← Theming, layout, animations
├── script.js         ← All data (projects, skills, notes)
└── assets/
    └── Adityaimg.png ← Profile photo
```

## Run Locally
Open `index.html` in browser, or use VS Code Live Server extension.

## Deploy (GitHub Pages)
```bash
git init
git add .
git commit -m "🚀 Launch portfolio"
git branch -M main
git remote add origin https://github.com/aditya018s/aditya018s.github.io.git
git push -u origin main
```
Then: GitHub repo → Settings → Pages → Source: main → Save.
Live in ~2 minutes at https://aditya018s.github.io
